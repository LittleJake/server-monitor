基于ThinkPHP5.1的服务器监控平台，需要配合服务端python脚本，使用Redis存储相关数据。

### 使用

#### 安装

```bash
git clone https://github.com/LittleJake/server-monitor
cd server-monitor
composer install
cp .env.example .env
```

#### API上报节点密钥操作（增/删/查）（TODO）

```bash
php think token help
```

#### 配置Redis数据源

```bash
vim .env
```

#### 刷新数据缓存

```bash
php think refresh:cache
```
